Informasi XIDZs-WRT

  • Detail Sistem:XIDZs-WRT Adalah Firmware Custom Berbasis OpenWrt Dan ImmortalWrt
  • Versi Firmware:23.05.x | 24.10.x | 25.12.x
  • Alamat Ip Default:IP: 192.168.1.1
  • Password Default:User: root | Pass: xyra
  • WIFI Jaringan:SSID: XIDZs / XIDZs_5G | Sandi: XIDZs2026
  • Instruksi GPIO:Aktifkan via /system/startup/rc.local (Edit file jika pin tidak cocok)
  • Perintah Terminal:Ketik xdev untuk XDEV Script Tools
  • Configurasi Bot:Configurasi via /usr/bin/xidz
  • Aturan Booting:Gunakan versi ModSdcard untuk emmc error dan normal ( jangan di install ke EMMC )
  • Saran Instalasi:Format Sdcard sebelum install atau burning, di sarankan gunakan Rufus/Etchdroid, lalu restart devices setelah booting awal agar configurasi sempurna
